## Markdown

### Basic syntax

Info here on data specs

Info here on more data specs

Info in **bold text**.

Info in *italicized text*.

Info in ***bold italics***.

> Block quoted text
>
> Block quoted paragraph 2

1. list item 1
2. list item 2
    1. nested item 1
    2. nested item 2
3. list item 3

- a note
- 2\. another note

    Paragraph under the note for explanation

- a third note
    - nested note
    - another nested note
- last note
